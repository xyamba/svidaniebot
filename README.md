## Шаг 1 — Получи токены

### Telegram Bot Token
1. Открой Telegram, найди @BotFather
2. Напиши `/newbot`
3. Придумай имя и username для бота
4. Скопируй токен вида: `7123456789:AAF...`

### Anthropic API Key (Claude AI)
1. Зайди на https://console.anthropic.com
2. Создай аккаунт и перейди в API Keys
3. Нажми "Create Key", скопируй ключ

### OpenWeatherMap API Key (погода — бесплатно)
1. Зайди на https://openweathermap.org/api
2. Зарегистрируйся → "My API Keys"
3. Скопируй ключ (активируется за 10 минут)

## Шаг 2 — Установка

```bash
# Установи зависимости
pip install -r requirements.txt
```

---

## Шаг 3 — Настройка

Отредактируй начало файла `date_bot.py` — замени токены:

```python
TELEGRAM_TOKEN = "твой_telegram_token"
ANTHROPIC_API_KEY = "твой_anthropic_key"  
WEATHER_API_KEY = "твой_openweather_key"
```

Или используй переменные окружения (рекомендуется):

```bash
export TELEGRAM_TOKEN="твой_токен"
export ANTHROPIC_API_KEY="твой_ключ"
export WEATHER_API_KEY="твой_ключ"
```

---

## Шаг 4 — Запуск

```bash
python date_bot.py
```

Открой бота в Telegram, напиши `/start` — и пользуйтесь вместе!

---

## Важно

- Бот работает для **двух людей** — оба пишут `/start` и получают доступ ко всем функциям
- Запланированное свидание видно обоим
- Для постоянной работы можно запустить на сервере (VPS) или через Railway/Render (бесплатно)
